การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์ Week # 2
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? (เชื่อม)
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? (ได้เรียนและทบทวน html จากความผิดพลาดและแก้งานส่งทำให้เข้าใจมากขึ้น)
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? (เข้าใจ html และ css จากนั้นนำมาใช้กับงานต่อไปจนสำเร็จ)
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? (ได้รู้ว่า CSS ไม่ยากและสามารถเรียนรู้ให้เชี่ยวชาญได้เหมือน html)
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? (ไปเรียนรู้เพิ่มเติมถ้ามีเวลาว่างพอ(อาทิตย์นี้งานล้นมือ))
